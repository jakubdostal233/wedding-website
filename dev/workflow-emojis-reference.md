# AI Agent Workflow Emoji Reference

The shared emoji vocabulary for this project. Emojis are welcome in `dev/` markdown and in chat with the owner (they help navigation); they are NOT used in `docs/`, in the website source (HTML/CSS/JS), or in config files. See `.claude/CLAUDE.md` &rarr; Conventions.

## Status / Workflow

| Emoji | Meaning | When to Use |
|---|---|---|
| ✅ | Completed / Working / Verified | Task finished successfully, component works correctly |
| ⚠️ | Caution / Partial Completion | Needs review, partially done, possible issue |
| ❌ | Failed / Broken / Blocked | Task failed, feature broken, cannot continue |
| 🔄 | In Progress / Updating | Running task, processing, retrying |
| ⏳ | Waiting / Queued / Pending | Blocked by dependency, scheduled for later |
| 🎯 | Goal / Objective | Define target, milestone, or objective |
| 🎉 | Success / Celebration | Milestone achieved, successful outcome |
| 🏆 | Winner / Best Choice | Best option selected, victory |
